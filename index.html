Einer der kleinsten In-App Router der Welt. $page.js ermöglicht dir problemlos dynamische Inhalte über die History API des Browsers zu steuern. Des Weiteren ist eine Mikro-Template Engine integriert, um Variablen aus dem JavaScript Context in deine HTML Seite einzubinden. $page.js wurde auf das Wesentliche optimiert und mit unter 1.9KByte lässt es sich einfach in vorhandene Websites integrieren.
