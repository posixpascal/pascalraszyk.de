Einige meiner Projekte

$page.js

Einer der kleinsten In-App Router der Welt. $page.js ermöglicht dir problemlos dynamische Inhalte über die History API des Browsers zu steuern. Des Weiteren ist eine Mikro-Template Engine integriert, um Variablen aus dem JavaScript Context in deine HTML Seite einzubinden. $page.js wurde auf das Wesentliche optimiert und mit unter 1.9KByte lässt es sich einfach in vorhandene Websites integrieren.

FormGenie

Eine drop-in JavaScript Bibliothek um Formulare zu validieren. FormGenie erkennt anhand des Input-Feldes bereits, welche Validierung zum Einsatz kommt. Nach dem Motto "Convention Over Configuration" erleichtert dir FormGenie das Arbeiten mit Formularen. Eine Beispiel-Seite befindet sich hier. FormGenie benötigt keine Abhängigkeiten, ist aber dennoch kompatibel zu den gägigsten JS Frameworks.

TeamBrewery

In meiner Freizeit spiele ich gerne eine kurze Partie Pokémon auf Pokemon Showdown. Mit TeamBrewery möchte ich der Community meinen Dank ausdrücken.
