AuthCtrl
controller in module frood.core

Dependencies

  • $scope
  • $state
  • $rootScope
  • $auth

Methods

  • authenticateWith(what, provider)

    Authenticate the user with the given provider (either facebook, twitter, google)

    Parameters
    ParamTypeDetails
    whatString

    A provider as string namely facebook, twitter or google.

    providerString

    A string with a valid (and already set-up) provider. we use facebook, twitter and google2-oauth

  • login(userForm)

    Logs the user in using the given credentials, retrieves Auth Tokens from webservice and closes the modal afterwards.

    Parameters
    ParamTypeDetails
    userFormObject

    logs the user in with the given credentials

  • register(userForm)

    Sends userForm to the $auth.submitRegistration function.

    Parameters
    ParamTypeDetails
    userFormObject

    an object containing the user informations

  • switchTab(newTab)

    Parameters
    ParamTypeDetails
    newTabString

    Either login or register which tab should be actively displayed in the auth modal

Properties

  • activeTab

    Used to indicate open tab in the auth-modal. (either login or register)

  • errors

    An array of strings representing error messages. this array is filled with server responses automatically.

  • isSignedIn

    Defines whether the user is signedIn or not. this is not reliable because the server checks the current users login state. You will get request-errors when playing with this var.
