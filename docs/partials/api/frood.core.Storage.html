Storage
service in module frood.core

Description

Wraps localStorage with a Service for easier access & a promise-able interface. This service will be changed in the future to store settings on the server side.

Dependencies

  • $q

Methods

  • get(key, defaultValue)

    Parameters
    ParamTypeDetails
    keyString

    the key that will be returned by the DB

    defaultValueObject

    a default value which will be used when the key doesn't exist

    Returns
    Promise

    a promise which will be resolved as soon as the value is returned.

    Example

    Storage.get("example").then(function(value){ => the value... });

  • set(key, value)

    Stores the given key to localStorage, returns a promise the promise will resolve after the key was set.

    Parameters
    ParamTypeDetails
    keyString

    the key that will be used as an identifier

    valueObject

    any JS object which will be stored

    Returns
    Promise

    a promise which will be resolved after the value is saved to localstorage

    Example

    Storage.set("example", [1, 2, 3, 4]).then(function(){ // do sth });

Example

Storage.set('key', { just: 'json'} ) // writes key to storage & serializeses JSON
Storage.get('key', [defaultValue]) // gets key from storage & parses JSON
