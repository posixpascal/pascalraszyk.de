APIEndpoint
service in module frood.core

Description

This function takes a segment & options and returns an URL

Dependencies

  • $location

Usage

APIEndpoint(segment, options);

Parameters

ParamTypeDetails
segmentString

URL to request from the API (eg: recipes/1.json)

optionsObject

define options like baseurl & prefix

Returns

String

an API url

Example

APIEndpoint("/recipes.json") => http://localhost:3000/api/v1/recipes.json APIEndpoint("/version", { prefix: ""}) => http://localhost:3000/version APIEndpoint("/index", {baseurl: "http://google.de"}) => http://google.de/api/v1/index
